Course Descriptions

Introduction to Computer Science
This course covers the fundamental concepts of computer science. Topics include algorithms, data structures, software engineering, and computer systems. Prerequisite: None Credits: 3
Web Development
An introduction to web development using HTML, CSS, and JavaScript. Students will learn how to create responsive and interactive websites. Prerequisite: Introduction to Computer Science Credits: 4
Data Structures and Algorithms
This course focuses on data structures such as arrays, linked lists, stacks, queues, trees, and graphs. Algorithms for searching, sorting, and optimization are also covered. Prerequisite: Introduction to Computer Science Credits: 4
Database Management Systems
This course provides an introduction to database management systems. Topics include database design, SQL, and transaction management. Prerequisite: Data Structures and Algorithms Credits: 3
